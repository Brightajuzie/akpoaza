# Handyman E-Commerce

## Run the app

Use the workspace root to start the frontend or backend:

- `npm run start:frontend` - start the Expo frontend
- `npm run start:backend` - start the backend API server
- `npm run start` - alias for `npm run start:frontend`

## Notes

- The Expo app lives in `frontend/`
- The Express backend lives in `backend/`
- Do not run `npx expo start` from the root directory; use the `frontend/` folder or the root scripts above
